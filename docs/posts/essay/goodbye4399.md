# 再见

## （一）

&emsp;&emsp;两年前，怀着忐忑不安的心情迎来了毕业，在校的第一次项目组面试并不顺利。刚来的三个月培训，高傲而又自卑，因为一直没找到项目组就焦虑着，于是当时就有了逃跑的想法。不过在HR的劝说下和中心老师们的鼓励下，最终成功进入了手游七。

&emsp;&emsp;进入项目组后刚开始也是各种紧张，担心自己能力问题，但是后来逐渐上手，业务娴熟，放开自我，最终和大家打成了一片。虽然相处还没有一年，项目组整体也比较低气压，但是相对来说，我更多的仍然是开心，大概是一种归属感的开心。

&emsp;&emsp;在后面经历了项目组的解散，以及情感的打击，那段时间沉沦而迷茫。但是在大家的鼓励下，我走出了那段最阴霾的时光。

&emsp;&emsp;兜兜转转又回到了中心，UGC项目让我开始思考我大学缺失的技术栈问题，于是自己也开始尝试别的东西，同时项目也给了我尝试别的技术的机会。在之后某一天也鼓起勇气和老师说了我的真实想法，并没有想象的任何担忧，老师是支持的，甚至是鼓励的。虽然最后没机会看到了 UGC 的第一个完整版本，但是，未来祝好！
